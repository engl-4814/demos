Kittens in the <img> element

The Basics

Below, I use one of my image files to render an image with the src, width, height, and alt attributes.

NOTE: We will only use the width and height attributes right now, since we have not covered CSS. After we learn CSS, you should definitely alter an image's size with CSS instead.

Using srcset & sizes

In the img element below, I use a set of the same 3 images, but differently sized, by using the srcset and sizes attributes. Use the:

  • w (width) value for the srcset, and
  • In sizes:
    1. Within parantheses, the max-width or min-width query attribute that uses the length value, such as px (pixel) in its query.
    2. After the paranthentical query, add a length value, such as a px value, for the image to be, when the condition has been met. 
<img srcset="images/cute-kittens-480.jpg 480w,
          images/cute-kittens-960.jpg 960w,
          images/cute-kittens-1440.png 1440w"
  sizes="(max-width: 480px) 320px,
         (max-width: 960px) 480px,
         (max-width: 1440px) 960px,
         1440px"
  alt="Five cute kittens set in a row on some green grass"   
/>

Use the inspection tool to review the syntax, or, if you have cloned this repo, open and review it in your code editor.

Kittens in the <picture> element

Below, I use the newer picture element that uses some simpler and more readable syntax:

  1. uses the standard <img> as the default use of the smallest image file.
  2. but <picture> also uses the <source> element to tell the browser to use the next largest image at a specified minimum width (px) of your choice.
  3. but the <picture> uses the <source> element to tell the browser to otherwise use the cropped version of your medium-sized file. 
<picture>
  <source media="(max-width: 480px)" 
          srcset="images/cute-kittens-135-cropped.jpg">
  <source media="(min-width: 960px)" 
          srcset="images/cute-kittens-960.jpg">
  
  <img src="images/cute-kittens-480.jpg"
        alt="Only 1 of 5 kittens shown in this cropped image *frowny face*" />
</picture>
