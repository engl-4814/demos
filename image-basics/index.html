The Basics

Below, I use one of my image files to render an image with the src, width, height, and alt attributes.

NOTE: We use the width and height attributes to tell the browser to reserve loading space. We do not use it to for styling/design. After we learn CSS, we can design flexible sizing of images with CSS.

On srcset

In the img element below, I use the srcset attribute to provide the browser a set of images to choose from for the user, based on factors like their device width, internet connection, and pixel density of the screen.

Each image source in the srcset needs 3 values separated by a comma and all contained within double-quotation marks:

  1. An image path to the file,
  2. a space, and
  3. The image's intrinsic width in pixels (w) or pixel density (x)
